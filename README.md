# Dice-Roller



A simple, interactive virtual dice-rolling web application built using vanilla JavaScript, HTML5, and CSS3. Click the roll button to generate a random number from 1 to 6 and watch the dice face dynamically update!

---

##  Features

* **Random Number Generation:** Uses JavaScript's math utilities to simulate a fair 6-sided die roll.
* **Dynamic Image Swapping:** Utilizes a `switch` statement to change the dice graphic layout instantly based on the result.
* **Clean Text Feedback:** Displays the rolled number in clear text alongside the visual graphic.
